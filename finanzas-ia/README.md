# Finanzas IA

App personal de finanzas en formato mobile/PWA.

## Qué incluye

- Registro de ingresos y egresos.
- Carga por texto natural.
- Dictado por voz si el navegador lo permite.
- Balance mensual.
- Ritmo del mes.
- Presupuestos por categoría.
- Gráficos y diagnóstico.
- Exportación CSV.
- Datos guardados localmente en el dispositivo.
- PWA instalable en iPhone/Android.

## Uso local

```bash
npm install
npm run dev
```

## Deploy gratis recomendado

### Opción A: Vercel

1. Subí esta carpeta a un repositorio de GitHub.
2. Entrá a Vercel.
3. Importá el repositorio.
4. Framework: Vite.
5. Build command: `npm run build`.
6. Output directory: `dist`.

### Opción B: Netlify

1. Subí esta carpeta a GitHub.
2. Entrá a Netlify.
3. Importá el repositorio.
4. Build command: `npm run build`.
5. Publish directory: `dist`.

## Instalar en iPhone

1. Abrí la URL en Safari.
2. Tocá compartir.
3. Elegí "Agregar a pantalla de inicio".
4. Nombre: Finanzas IA.

## Nota de seguridad

La app guarda datos en el navegador del celular. Exportá CSV periódicamente como respaldo.
